## Access model

This repository is **public** but **unlisted** — it lives at a long, obscure URL and is not advertised, indexed, or promoted. Anyone with the link can read it; the link itself is the access control. Please don't post the URL publicly.

## Using this wiki to draft content

For LLMs and humans drafting posts, captions, emails, or talks:

1. Read [`wiki/CLAUDE.md`](CLAUDE.md) first — it's the schema + operating guide.
2. Open [`wiki/index.md`](wiki/index.md) to locate relevant articles.
3. Read the articles in full.
4. Quote Ra directly where precision matters. Paraphrase in the same register.
5. **Never invent quotes.** If a phrase isn't in a sourced article, don't attribute it to Ra.
6. Run your draft past the "do not" list in [`wiki/voice/voice-guide.md`](wiki/voice/voice-guide.md) before finalizing.

## Contributing

See [`CLAUDE.md`](CLAUDE.md) for the full schema and workflows. Short version:

- LLM-maintained under supervision. Ask for updates; the LLM edits articles and the index in the same pass.
- Don't hand-edit unless you're certain about the material and the voice.
- Treat anything private (individual mentees, internal business data, closed-door material) as off-limits.
